Overview
Smart Property Advisor is a machine learning–based application that helps users estimate property prices based on different property features. Users can enter details such as area, number of bedrooms, location, and amenities, and the system predicts an estimated property price.
The project uses data processing and machine learning to assist users in making better real estate decisions through an easy-to-use web interface.
Features
Property price prediction using machine learning
Interactive web application built with Streamlit
Automatic dataset generation
Model training and saving
Data visualization using Plotly
Optional API support for backend services
Technologies Used
Python
Streamlit
Pandas
NumPy
Scikit-learn
Plotly
FastAPI
How to Run
1. Install Dependencies

pip install -r requirements.txt
2. Setup the Project
This will create folders, generate data, and train the model.

python run.py --setup
3. Run the Application

python run.py
The Streamlit application will start and open in your browser.
